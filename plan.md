# MySQL内核理解课程 - 完整计划

## 项目概述

本项目将爬取阿里团队维护的数据库内核博客（http://mysql.taobao.org/monthly/）的所有文章，结合MySQL源码版本，重新梳理成一个系统化的MySQL内核理解课程。

## 一、网站分析

### 1.1 网站结构
- **主页**: http://mysql.taobao.org/monthly/ - 列出所有月份的索引
- **月份页面**: http://mysql.taobao.org/monthly/YYYY/MM/ - 列出该月份的所有文章
- **文章详情**: http://mysql.taobao.org/monthly/YYYY/MM/DD/ - 具体文章内容

### 1.2 时间范围
- 最早文章: 2014年8月
- 最新文章: 2026年3月
- 约139个月的文章

## 二、爬虫方案设计

### 2.1 爬虫工具选择
- 使用Python + requests + BeautifulSoup4
- 异步请求，尊重robots.txt
- 设置合理的请求延迟

### 2.2 爬虫流程
1. 爬取主页，获取所有月份链接
2. 逐个爬取月份页面，获取文章链接
3. 逐个爬取文章详情页
4. 保存文章内容（HTML和元数据

### 2.3 数据存储结构
```
data/
├── articles/
│   ├── 2014/
│   │   └── 08/
│   │       ├── 01.md
│   │       └── metadata.json
│   ├── 2015/
│   └── ...
└── index.json
```

## 三、MySQL源码版本对应关系

### 3.1 版本对应策略
- 根据文章发布时间对应MySQL大版本
- 主要版本: 5.5, 5.6, 5.7, 8.0, 8.1, 8.2, 8.3, 8.4
- 使用GitHub官方源码仓库: https://github.com/mysql/mysql-server

### 3.2 源码获取策略
- 使用Git子模块管理MySQL源码
- 按版本标签检出对应源码
- 建立文章与源码版本的映射关系

## 四、课程内容结构设计

### 4.1 课程组织方式
按技术主题分类，而非时间顺序

### 4.2 课程模块
1. **存储引擎模块**
   - InnoDB架构
   - 缓冲池管理
   - 事务与锁
   - 索引结构

2. **查询优化器模块**
   - 查询解析
   - 执行计划
   - 索引选择
   - 成本估算

3. **复制与高可用模块**
   - Binlog机制
   - 主从复制
   - 半同步复制
   - Group Replication

4. **性能优化模块**
   - 参数调优
   - 性能诊断
   - 锁优化
   - 索引优化

5. **新版本特性模块**
   - 8.0新特性
   - 8.4 LTS特性
   - 性能改进

### 4.3 单篇课程结构
```
课程标题
├── 1. 文章概述
├── 2. 原文链接
├── 3. 核心知识点
├── 4. 源码解析
│   ├── 文件路径
│   ├── 关键函数
│   ├── 代码片段
├── 5. 实践示例
├── 6. 总结
└── 7. 扩展阅读
```
